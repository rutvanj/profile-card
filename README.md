# Profile Card

A simple and responsive profile card built using **HTML and CSS**.
This project demonstrates basic frontend layout techniques including card design, gradient backgrounds, and centered positioning.

## Features

* Clean and minimal UI design
* Gradient background styling
* Responsive centered card layout
* Simple and lightweight implementation

## Technologies Used

* HTML
* CSS

## Project Purpose

This project was created to practice fundamental **frontend development concepts** such as layout design, styling with CSS, and building simple UI components.

## How to Use

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
